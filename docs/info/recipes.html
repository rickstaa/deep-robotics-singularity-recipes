NVIDIA container recipes¶

tf-gpu-conda3-ros_kinetic-moveit-cuda10-xenial¶

This recipe sets up a UBUNTU 16.04 (xenial) container with the following packages installed in it:

  • ROS kinetic

  • moveit

  • Latest stable tensorflow GPU release

  • Miniconda3

  • Cuda 10

  • Cudnn 7
